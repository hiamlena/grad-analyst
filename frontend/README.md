# Frontend MVP (статичный прототип)

Это первый статичный экран сервиса «ГрадоАналитик» по UI-спецификации.

## Что внутри

- `index.html` — структура экрана и все блоки MVP-формы.
- `styles.css` — базовые стили делового интерфейса и адаптивная сетка.

## Как запустить локально

1. Откройте папку `frontend/`.
2. Дважды кликните `index.html` **или** откройте файл в браузере через `File → Open`.

## Ограничения прототипа

- Это только визуальный прототип без backend.
- Реальные загрузки файлов на сервер пока не подключены.
- Интеграции с OpenAI, БД и внешними сервисами отсутствуют.
