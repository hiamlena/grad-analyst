# Шаблон итогового отчёта

Итоговый отчёт должен содержать следующие разделы:

1. Краткий вывод
2. Исходные данные
3. Загруженные документы
4. Территориальная зона
5. Текущий ВРИ
6. Допустимые ВРИ
7. Проверка желаемого объекта
8. Параметры застройки
9. Ограничения
10. Риски
11. Рекомендации
12. Предварительный 2D-эскиз
13. Что нужно запросить дальше
14. Уровень уверенности
15. Источники
