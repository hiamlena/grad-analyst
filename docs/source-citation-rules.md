# Правила ссылок на источники

- ИИ не может делать вывод без источника, если источник нужен.
- Для каждого вывода нужно указывать:
  - документ;
  - страницу или раздел, если известно;
  - краткий фрагмент-основание;
  - уровень уверенности.
- Если источник не найден, нужно писать:

«Источник не найден в загруженных документах».

- Если данных недостаточно, нужно писать:

«Данных недостаточно. Для вывода нужно загрузить: ...»
