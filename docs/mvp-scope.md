# MVP Scope — ГрадоАналитик

## Что входит в первый шаг

- Минимальный каркас папок проекта.
- Документ с границами MVP.
- Документ с контрактом достаточности входных данных.
- Базовый промпт анализа с запретом на выдумывание данных.

## Что не входит в этот шаг

- Next.js / FastAPI / любая серверная или клиентская инфраструктура.
- Подключение OpenAI API.
- Подключение SQLite или другой БД.
- Карты, Росреестр, GIS, авторизация.

## Цель шага

Зафиксировать структуру и правила MVP до начала реализации кода.
