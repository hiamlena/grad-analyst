# ГрадоАналитик

MVP ИИ-сервиса для предпроектного анализа земельного участка.

## Цель

Пользователь вводит данные участка, загружает документы ПЗЗ, ГПЗУ, ЕГРН и получает:

- предварительный градостроительный анализ;
- вывод: можно / нельзя / возможно при условиях / данных недостаточно;
- риски и ограничения;
- рекомендации;
- предварительный 2D-эскиз размещения объекта.

## Первый MVP

В первую версию входит:

- один пользователь;
- ручной ввод кадастрового номера, адреса и параметров участка;
- загрузка документов PDF/DOCX/TXT;
- извлечение текста из документов;
- ИИ-анализ через RAG;
- отчёт;
- простой SVG-эскиз.

## Не входит в MVP

- интеграция с Росреестром;
- база всех ПЗЗ Краснодарского края;
- сложная GIS-карта;
- 3D-визуализация;
- юридически значимое заключение.

## Стек

- Next.js
- TypeScript
- SQLite на старте
- OpenAI API
- SVG для эскизов
